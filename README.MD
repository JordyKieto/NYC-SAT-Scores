# NYC SAT DATA

## Goal
*  Provide analysis and visualization of SAT scores in New York city dataset

## How to Run
### *Server
* run 
    >pip install -r requirements.txt
* run 
    >FLASK_APP=app.py flask run
### *Client
* cd into client directory
* run 
    >npm install
* run 
    >npm start

## Technologies Used
* Pandas 
* Flask
* React / Redux
